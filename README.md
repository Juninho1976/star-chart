# star-chart
Creating a star chart app with the help of Chat GPT... ! This will eventually be publicly hosted I think... 



## Local development
cp infra/.env.example infra/.env
docker compose --env-file infra/.env up --build



Web:
	•	http://localhost:3000

API:
	•	http://localhost:8080

Viewer accounts:
	•	Read-only access
	•	Can view stars & rewards
	•	Cannot modify data
